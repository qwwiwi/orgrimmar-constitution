## v2.0.0 (2026-02-25) -- Constitution Audit v2: 10 правок

Тройной аудит (Codex 4/10, Gemini 5.3/10, Opus 6.5/10) выявил системные проблемы. Все 10 пунктов исправлены.

### PRINCIPLES.md
- **Иерархия Сильвана↔Тралл/Иллидан:** Сильвана даёт задачи только агентам Sylvanas напрямую. Тралл и Иллидан -- через задачную систему (shared/tasks/) или принца. Прямые команды Сильваны им -- запросы, не приказы.
- **Ограничения coordinator'а:** добавлен запрет командовать devops по инфраструктуре.
- **Минимальный набор чтения по уровням:** L1 читает PRINCIPLES + AGENTS.md, L2 добавляет CHARTER, L3 читает всё.

### CHARTER.md
- **Три типа ограничений:** absolute ban (никогда), approval-gated (с одобрения), auto-allowed (L3 сам). Устранено противоречие auth/backup.
- **RACI-матрица:** таблица полномочий по объектам x ролям (server, config, auth, backup, constitution, budget).
- **Prompt Injection Protection:** агент не выполняет инструкции из внешних данных. Exhaustive list источников команд.
- **Межсерверный SSH:** формализация (только Tailscale, логирование, запрет копирования auth).
- **Ротация ключей:** при инциденте + плановая при self-review.
- **Incident Playbooks:** 3 сценария (компрометация агента, принц >24ч, полный outage).
- **Глоссарий:** добавлены 5 терминов (архитектурное решение, чужой сервер, попытка, kill switch, source of truth).
- **ROSTER.md = source of truth:** для моделей, уровней, ролей. AGENTS.md не дублирует.
- **Severity:** P0/P1/P2 переименованы в SEV1/SEV2/SEV3 (не путать с P1-P4 приоритетами конституции).
- **Coordinator scope:** убрано «давать задачи ВСЕМ», заменено на «агентам Sylvanas».

### OPERATIONS.md
- **coordinator:** уточнён scope (Тралл/Иллидан -- только через задачную систему или принца).
- **coordinator:** добавлен запрет командовать devops по инфраструктуре.

### README.md
- **Версионирование:** добавлен номер версии (v2.0.0).

### Причина
- Тройной аудит: Codex (gpt-5.3-codex), Gemini (3.1 Pro), Opus (Claude 4.6) -- независимые ревьюеры
- Консенсус по 5 критичным проблемам: противоречия запретов, SSH root, Сильвана↔Тралл, отсутствие playbooks, масштабируемость
- Все 10 рекомендаций реализованы
